# Flappy Bird game using DQN
This project involves training the bird in flappy bird to fly in between the pipes.
We can also train the cartpole using the same code.
you can tweak the different hyperparameters in the hyperparameters.yml file.

### Train and run flappy bird
- To train flappy bird, run this command: python agent.py flappybird1 --train
- To run, we just remove --train

### Train and run cartpole
- To train the carpole, run this command: python agent.py cartpole1 --train
- To run, we just remove --train
